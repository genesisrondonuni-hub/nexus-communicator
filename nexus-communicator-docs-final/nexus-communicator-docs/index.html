Instalación y Configuración

Requisitos del Sistema

Frontend (Netlify)
  • React 18+
  • Node.js 16+
  • Navegador moderno
Backend (Servidor)
  • Python 3.8+
  • Flask 2.0+
  • SQLite/PostgreSQL

Pasos de Instalación

1. Descargar Archivos

Descarga los archivos de la aplicación desde el repositorio.

git clone https://github.com/tu-usuario/nexus-communicator.git
2. Configurar Backend

Instala dependencias y configura la base de datos.

cd nexus-communicator-backend
pip install -r requirements.txt
python src/main.py
3. Desplegar Frontend

Construye y despliega en Netlify.

cd nexus-communicator-frontend
npm install
npm run build

Preguntas Frecuentes

¿Cómo obtengo las API keys necesarias?

Para WhatsApp Business API, regístrate en Meta for Developers. Para Gmail, usa Google Cloud Console. Para Gemini, accede a Google AI Studio. Todas las guías detalladas están en la sección de tutoriales.

¿Puedo importar contactos desde Excel?

Sí, la plataforma soporta importación desde archivos Excel (.xlsx), CSV y Google Sheets. Usa los botones "Importar de Sheets" e "Importar de Drive" en la sección de contactos.

¿Cómo funciona la automatización con IA?

Gemini 2.5 Pro analiza los mensajes entrantes y responde automáticamente basándose en tu base de conocimiento. Configura la información de tu negocio en Configuración > Automatización para mejores respuestas.

¿Qué tipos de multimedia puedo enviar?

Puedes adjuntar imágenes, videos, documentos PDF, archivos de audio y documentos de texto. El tamaño máximo por archivo es de 16MB para WhatsApp y 25MB para email.

Solución de Problemas

Errores Comunes

Error de API Key inválida

Verifica que las API keys estén correctamente configuradas y tengan los permisos necesarios.

Mensajes no se envían

Revisa la configuración de WhatsApp Business API y que el número esté verificado.

IA no responde automáticamente

Asegúrate de que la automatización esté activada y la base de conocimiento configurada.

Soluciones Rápidas

Reiniciar conexiones

Ve a Configuración > API Keys y vuelve a guardar las credenciales.

Limpiar caché del navegador

Presiona Ctrl+Shift+R (Windows) o Cmd+Shift+R (Mac) para recargar completamente.

Verificar estado del servicio

Consulta el dashboard para ver el estado de las conexiones y servicios.
